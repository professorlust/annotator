# Annotator
A Crowdsourcing NLP Annotation Platform with Multiple Quality Control Mechanisms powered by Tornado and MongoDB.

## Environment

- python 3.6.2
- tornado 4.5.2
- pymongo 3.6.1
- MongoDB 3.6

## Usage

```shell
> cd /path/to/annotator
> python3 annotator.py
```
